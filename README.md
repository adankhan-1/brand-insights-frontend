# Brand Insights Frontend

This is the React-based frontend for the Brand Insights application.

---

## Prerequisites

- Node.js (version 16 or higher recommended)
- npm (comes with Node.js)

---

## Setup & Run Locally

1. **Clone the repository**

git clone <your-repo-url>

2. **Install dependencies**

npm install

3. **Configure environment variables**

Create a .env file in the root directory with REACT_APP_BACKEND_URL variable.

4. **Start the development server**

npm start
